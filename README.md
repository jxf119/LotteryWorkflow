# 🎯 大乐透预测自动化系统

基于Dify工作流的大乐透彩票预测自动化系统，实现数据采集、智能预测、结果通知的完整流程。

## 🚀 功能特性

- **📊 数据采集**: 自动从500彩票网抓取最新开奖数据
- **🧠 智能预测**: 基于历史数据的机器学习预测算法
- **📱 多渠道通知**: 支持Slack、飞书、邮件通知
- **🔧 可配置**: 灵活的配置文件系统
- **📈 监控告警**: 系统运行状态实时监控

## 🛠️ 技术架构

```
dify_workflow/
├── src/                    # 源代码目录
│   ├── services/          # 业务服务层
│   ├── models/            # 模型层
│   ├── database/          # 数据层
│   ├── utils/             # 工具类
│   └── workflows/         # Dify工作流定义
├── config/                # 配置文件
├── tests/                 # 测试文件
├── lottery_system.py      # 主系统入口
└── requirements.txt       # 依赖列表
```

## 🎯 快速开始

### 1. 环境准备

```bash
# 安装依赖
pip install -r requirements.txt

# 初始化数据库
python src/database/init_db.py
```

### 2. 配置系统

编辑 `config/config.yaml` 文件配置相关参数

### 3. 运行系统

```bash
# 验证系统功能
python verify_system.py

# 运行完整工作流
python lottery_system.py
```
